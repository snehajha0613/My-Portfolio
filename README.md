# 🌐 Sneha Jha | Developer Portfolio

Hi! I'm Sneha 👋
This is my personal portfolio website where I showcase my work, skills, and journey as a Frontend Developer.

I built this project using React to create a fast, modern, and responsive experience that reflects both my technical abilities and design sense.

🔗 **Live Website:** https://my-portfolio-ten-rho-59.vercel.app/

---

## ✨ What you’ll find here

* A clean and modern UI with a dark theme and glass-like effects
* Smooth animations that make the experience feel interactive and alive
* A responsive layout that works across mobile, tablet, and desktop
* A projects section highlighting my key work and learning
* Sections for my education, certifications, and contact details

---

## 🛠️ Tech I used

* React.js
* JavaScript (ES6+)
* CSS3 + Bootstrap
* Framer Motion (for animations)
* React Icons
* Deployed using Vercel

---

## 📁 Project structure

```
src/
  components/      → All UI sections (Navbar, Hero, Projects, etc.)
  assets/          → Images and files used in the project
  App.js           → Main structure of the app
  App.css          → Styling

public/
  certificates/    → My certificates
  index.html       → Entry point
```

## 🚀 Run locally

If you want to run this project on your system:

```
git clone https://github.com/snehajha0613/your-repo-name.git
cd your-repo-name
npm install
npm start
```

Then open: http://localhost:3000
